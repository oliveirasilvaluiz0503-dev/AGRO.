# AgroForte Sustentável 🌾🚜

> Um projeto dedicado a promover o equilíbrio essencial entre a alta produtividade agrícola e a preservação do meio ambiente através da tecnologia e de boas práticas de manejo.

Este repositório contém a interface de um site institucional moderno e uma ferramenta de back-end em Java desenvolvida para calcular o índice de sustentabilidade de propriedades rurais.

---

## 🚀 Funcionalidades

- **Interface Responsiva:** Landing page adaptável para dispositivos móveis, tablets e computadores.
- **Identidade Visual Temática:** Design focado em tons naturais (verdes e terrosos) que refletem o agronegócio moderno.
- **Motor de Cálculo (Back-end Java):** Algoritmo que analisa o percentual de áreas preservadas e o uso de rotação de culturas para classificar o nível ecológico da fazenda.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias:

* **HTML5:** Estrutura semântica das páginas web.
* **CSS3:** Estilização, layout flexível (Flexbox) e responsividade (Media Queries).
* **Java:** Lógica de negócio no back-end para processamento e classificação de dados de sustentabilidade.
